Уроки Java

Урок 1

Програмування на Java
Ми будемо вивчати Java для початківців і вивчимо стандартні речі, ООП, а також торкнемося теми пов'язані тільки з Java. Отже, приступаємо до вивчення мови програмування Java!

Урок 2

Установка Java JDK і IntelliJ IDEA
Без JDK ми не зможемо нормально працювати, а без середовища розробки як IntelliJ IDEA нам буде ніде писати код, тому ми повинні встановити обидві ці речі.

Урок 3

Hello World!
Нарешті ми з вами добралися до програмування. У цьому уроці ми з вами розберемося з нашою програмою, а також напишемо невелику програму, яка буде виводити в консоль вітання "Hello World!".

Урок 4

Змінні
У цьому уроці ми з вами вивчимо змінні в мові Java. Змінні це неймовірно важлива тема. Завдяки змінним ми можемо зберігати будь-яку інформацію, а потім користуватися цією інформацією.

Урок 5

Дані від користувача
У цьому уроці ми навчимося приймати дані від користувача, обробляти їх і робити з ними все що завгодно. Ми будемо використовувати для цих цілей клас Scanner.

Урок 6

Математичні операції
У цьому уроці ми з вами розглянемо всі основні математичні операції в мові Java. Ми навчимося працювати з цифрами, а саме збільшувати їх або ж зменшувати.
